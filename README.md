# 🚦 Semáforo com Arduino

Projeto simples de semáforo utilizando Arduino e protoboard.

## 🔧 Componentes
- Arduino
- LEDs (vermelho, amarelo, verde)
- Resistores
- Protoboard
- Jumpers

## 💻 Funcionamento
O sistema simula um semáforo com temporização entre os LEDs, controlado por programação em C no Arduino.

## 📌 Conceitos aplicados
- Lógica de programação
- Temporização (delay)
- Circuitos eletrônicos básicos
